# ScrapySpider_YangGuang
本项目用于爬取「东莞阳光热线问政平台」中的投诉标题、申诉部门、处理状态、发布时间、详情页面中的投诉文本内容以及图片并存储到MongoDB中。

实战Scrapy之阳光政务平台爬虫项目
【项目描述】：爬取「东莞阳光热线问政平台」中的投诉标题、申诉部门、处理状态、发布时间、详情页面中的投诉文本内容以及图片。
